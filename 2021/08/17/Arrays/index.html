Arrays

2021-08-17

Arrays类

Arrays类是包含在java.util包中的一个方便对数组操作的类，其中包含升序，降序排列以及查询等操作。

Arrays类常用方法

boolean equals (array1,array2)

各种类型数组均可使用，下面以boolean数组为例：
public static boolean equals(boolean[] a,
boolean[] a2)

比较array1和array2是否相等。如果两个指定的 boolean 型数组彼此 相等，则返回 true。如果两个数组包含相同数量的元素，并且两个数组中的所有相应元素对都是相等的，则认为这两个数组是相等的。换句话说，如果两个数组以相同顺序包含相同的元素，则两个数组是相等的。此外，如果两个数组引用都为 null，则认为它们是相等的。
eg：int a = boolean equals (array1,array2);

sort(array)

public static void sort(arrays[] a)
对指定的特定类型arrays数组按数字升序进行排序。
eg:sort(arrya1);

toString

public static String toString(boolean[] a)
返回指定数组内容的字符串表示形式。字符串表示形式由数组的元素列表组成，括在方括号（ “[]”）中。相邻元素用字符 “, “（逗号加空格）分隔。这些元素通过 String.valueOf(boolean) 转换为字符串。如果 a 为 null，则返回 “null”。

fill

public static void fill(arrays[] a, val)
将指定的 val 值分配给指定 arrays 数组的每个元素。

copyOf

public static int[] copyOf(int[] original,
int newLength)
复制指定的数组，截取或用 0 填充（如有必要），以使副本具有指定的长度。对于在原数组和副本中都有效的所有索引，这两个数组将包含相同的值。对于在副本中有效而在原数组无效的所有索引，副本将包含 0。当且仅当指定长度大于原数组的长度时，这些索引存在。
即如果指定新数组长度要大于原数组长度，新数组中空值用0填充。新数组的长度小于原数组的话，只拷贝最前面的数据元素。
参数：
original - 要复制的数组
newLength - 要返回的副本的长度

binarySearch

public static int binarySearch(int[] a,int fromIndex,int toIndex,int key)

使用二分搜索法来搜索指定的 int 型数组的范围，以获得指定的值。返回下标。
必须在进行此调用之前对范围进行排序（通过 sort(int[], int, int 方法）。
如果没有对范围进行排序，则结果是不确定的。如果范围包含多个带有指定值的元素，则无法保证找到的是哪一个。

eg: int index=binarySearch(int[] a,val)
找到val在数组中的位置并返回其下标。
